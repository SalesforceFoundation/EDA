Address Management Related Classes

Address Management is made up of several components:

  • Multiple Address objects supported on Accounts
  • Seasonal Address support

Classes involved with Address object support:

  • ADDR_Account_TDTM - Trigger Handler on Account for Address management. Creates Address__c record from Account.
  • ADDR_Addresses_TDTM - Trigger handler on Address object to support propagating addresses.
  • ADDR_Contact_TDTM - Creates Address__c record from Contact.

Classes involved with Seasonal Address support:

  • ADDR_Seasonal_BATCH - Updates seasonal addresses in batch.
  • ADDR_Seasonal_SCHED - Supports the automatic scheduling of Seasonal Address updates.
